Around the Web

CoinGossip

With the increased interest in cryptocurrencies, this tool makes it easier to keep track of the discussions and news about various crytocurrencies. Data mining is done 24/7 to track discussions on various communities, with algorithms applied to provide for automated sentiment analysis.

This site has been featured on various CryptoCurrency blogs, such as The Merkle and BitcoinExchangeGuide.

  • Website: Link
  • Language: Python + HTML
  • Date: February 2018
  • Type: Cryptocurrency, Data Mining, Data Analytics

Smart Mirror

Inspired by the new wave of Smart Mirrors, this serves as the software layer that includes various useful information, including weather, news tickers and stock & cryptocurrency tickers. By being a webapp, this can be easily deployed to any DIY smart mirror installation, as well as Android kiosk devices.

  • Demo: Link
  • Language: HTML5 + jQuery
  • Date: December 2017
  • Type: Smart Home

Alastair

Alastair is a home automation system that uses intelligent sensor alogrithms to control lighting and cooling systems. It is a Python program that is suitable for use in low-power headless devices, such as the Raspberry Pi.

  • Language: Python
  • Date: March 2016
  • Type: Smart Home

Nearly

Nearly (stylised: NEArly) is a minimalist weather app that provides hyper-accurate forecast for specific regions in Singapore, using data from NEA's 2 Hour Nowcast. It was designed to be used on kiosk devices, such as Android tablets.

  • Demo: Link
  • Language: HTML (jQuery)
  • Date: January 2016
  • Type: Smart Home

Relatus

Relatus is a bespoke resource management system that was custom-made for a Singaporean non-profit organisation. It was designed to fit their specific needs for shared information management (including data on clients, partners and training sessions), automated reporting and basic analysis capabilites.

  • Language: PHP, MySQL
  • Date: November 2015
  • Type: Enterprise Resource Planning

The Resistance

This is a web based renditon of the popular card game The Resistance, where players attempt to deduce one another's identities. This app replaces the need for game cards by utilising a single mobile phone that is shared among the players.

Timemoo

Timemoo is a time zone converter that process queries using natural language processing (NLP). It is also able to convert time from location to location, removing the need for specific time zones.

  • Date: June 2013
